# 雨巷纸伞 - 跑酷游戏

## 原始需求

> 扩展风向变化路段：巷口、桥面和高楼夹缝会周期性变风，开伞可以滑翔更远，也可能被吹偏撞上灯笼架；风铃、雨帘和伞面倾斜要提前提示玩家。
> 扩展纸伞耐久管理：暴雨、碰撞、强风和积水会降低伞面耐久，玩家需要在修伞摊、捷径、金币和连送时间之间做取舍，伞快破时操作手感也要有所变化。

## 游戏简介

《雨巷纸伞》是一款以江南雨巷为背景的跑酷游戏。玩家扮演一位在雨中老街递送纸伞的信使，需要穿过各种障碍物，同时保护纸伞不被损坏。

### 游戏特色

- **风向变化系统**：巷口、桥面、高楼夹缝会周期性变换风向，开伞可滑翔更远但可能被吹偏
- **纸伞耐久系统**：纸伞会被暴雨、强风、碰撞、积水等损耗，耐久归零则游戏结束
- **预告提示系统**：风铃预警强风，雨帘预警暴雨，伞面倾斜提前提示风力方向
- **多样地形障碍**：屋檐、水坑、风口、石桥、灯笼摊、码头台阶、狭窄巷道、修伞摊、捷径
- **丰富操作**：开伞滑翔、收伞冲刺、贴墙转弯、修补伞面、修伞摊停留修伞
- **天气系统**：小雨、中雨、大雨、暴雨，影响伞面损耗速度
- **收集系统**：收集油纸、蜡块和金币修复伞面或获得分数
- **取舍机制**：修伞摊恢复耐久但耽误时间，捷径加速但损耗伞面
- **连送奖励**：连续递送和时间奖励系统
- **操作手感变化**：伞快破时转向延迟、摇晃加剧、风力影响更大
- **水墨风格**：江南水乡的雨夜氛围

## 技术栈

- **PixiJS 7** - 2D WebGL 渲染引擎
- **Vite** - 前端构建工具
- **原生 JavaScript** - 游戏逻辑

## 启动方式

### 前置要求

- Node.js >= 16.0.0
- npm 或 pnpm 包管理器

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务器

```bash
npm run dev
```

访问地址：http://localhost:5173

#### 3. 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

#### 4. 预览生产版本

```bash
npm run preview
```

## Docker 一键启动

### 前置要求

- Docker
- Docker Compose

### 启动步骤

#### 1. 构建并启动容器

```bash
docker compose up --build
```

如需后台运行：

```bash
docker compose up --build -d
```

访问地址：http://localhost:5173

#### 2. 停止服务

```bash
docker compose down
```

## 游戏机制

### 风向变化路段

- **巷口**：周期性变换风向，风力中等，开伞获得加分但伞面轻微损耗
- **桥面横风**：风力较强，变换周期较长，开伞更容易被吹偏
- **高楼夹缝**：风力最强，变换周期短，需要快速反应

### 预告提示

- **风铃**：黄色警示，提示即将到来的强风区域
- **雨帘**：蓝色警示，会触发临时暴雨天气（5秒）
- **伞面倾斜**：屏幕底部指示器显示当前风力方向和强度，角色伞面会实时倾斜

### 纸伞耐久管理

不同来源对伞面耐久的差异化影响：

| 来源 | 开伞损耗 | 收伞损耗 | 说明 |
|------|---------|---------|------|
| 小雨 | 0.3/秒 | 0.8/秒 | 基础天气损耗 |
| 中雨 | 0.8/秒 | 2/秒 | 正常损耗速度 |
| 大雨 | 1.5/秒 | 4/秒 | 损耗较快 |
| 暴雨 | 3/秒 | 8/秒 | 损耗很快，需尽快避雨 |
| 水坑 | 3 | - | 踩水坑加速但轻微损耗 |
| 强风 | ×1.5 | ×0.8 | 开伞受风影响更大 |
| 碰撞 | 高 | 高 | 直接撞到障碍物 |

**低耐久手感变化**：
- 耐久 < 60%：警告变色
- 耐久 < 40%：转向延迟增加、摇晃加剧、风力影响增大
- 耐久 < 20%：严重危险，伞面剧烈摇晃，操作极度不稳定，耐久损耗加速

### 修伞摊与取舍机制

- **修伞摊**：按 E 键停留修伞，2秒内恢复 35 点耐久，但会耽误递送时间
- **捷径**：获得 3 秒 1.8 倍加速，但损耗 5 点伞面耐久
- **金币收集**：获得分数并累计连送次数
- **连送奖励**：连续修伞/收集金币获得额外加分，碰撞障碍物会重置连送
- **时间奖励**：每 10 秒递送时间获得 50 分奖励

### 障碍物

- **屋檐**：可以避雨，减少伞面损耗
- **水坑**：踩中可短暂加速，轻微损耗伞面
- **风口**：会推动角色并损伤伞面
- **石桥**：碰撞会损伤伞面并造成击退
- **灯笼摊**：开伞经过会碰撞，收伞可安全通过
- **码头台阶**：碰撞会损伤伞面
- **狭窄巷道**：开伞经过会碰撞，收伞可安全通过
- **巷口/桥面/高楼夹缝**：周期性变风，开伞加分但易被吹偏
- **风铃**：强风预警提示
- **雨帘**：暴雨预警，触发临时暴雨
- **修伞摊**：停留修伞恢复耐久
- **捷径**：加速但损耗伞面

### 收集品

- **油纸**：恢复 15 点伞面耐久，获得 100+ 连送奖励分
- **蜡块**：恢复 25 点伞面耐久，提供 5 秒防水保护，获得 150+ 连送奖励分
- **金币**：获得 200+ 连送奖励分，累计连送次数

### 天气系统

- **小雨**：伞面损耗较慢
- **中雨**：正常损耗速度
- **大雨**：伞面损耗较快
- **暴雨**：伞面损耗很快

## 注意事项

- 游戏使用三跑道横向移动设计
- 障碍物出现前会有对应颜色的预告提示（红色危险/黄色风警/蓝色雨警/绿色修伞）
- 难度会随时间逐渐增加
- R键修补伞面有 8 秒冷却时间
- E键在修伞摊范围内才会生效
- 伞面耐久低于 40% 时注意操作手感变化
