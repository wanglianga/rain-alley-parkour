# 雨巷纸伞 - 跑酷游戏

## 原始需求

> 请用 PixiJS 制作雨巷纸伞跑酷游戏，玩家在雨中老街递送纸伞，穿过屋檐、水坑、风口、石桥、灯笼摊、码头台阶和狭窄巷道。角色可以开伞滑翔、收伞冲刺、借屋檐避雨、踩水坑加速、贴墙转弯和短暂修补伞面，但纸伞会被强风、积水、碰撞和长时间暴雨影响耐久。玩法要有顺滑速度感、清楚障碍预告、收集奖励和失误恢复，玩家靠天气与地形即时判断，而不是死记整条路线。

## 游戏简介

《雨巷纸伞》是一款以江南雨巷为背景的跑酷游戏。玩家扮演一位在雨中老街递送纸伞的信使，需要穿过各种障碍物，同时保护纸伞不被损坏。

### 游戏特色

- **纸伞耐久系统**：纸伞会被暴雨、强风、碰撞等损耗，耐久归零则游戏结束
- **多样地形障碍**：屋檐、水坑、风口、石桥、灯笼摊、码头台阶、狭窄巷道
- **丰富操作**：开伞滑翔、收伞冲刺、贴墙转弯、修补伞面
- **天气系统**：小雨、中雨、大雨、暴雨，影响伞面损耗速度
- **收集系统**：收集油纸、蜡块和金币修复伞面或获得分数
- **水墨风格**：江南水乡的雨夜氛围

## 技术栈

- **PixiJS 7** - 2D WebGL 渲染引擎
- **Vite** - 前端构建工具
- **原生 JavaScript** - 游戏逻辑

## 启动方式

### 前置要求

- Node.js >= 16.0.0
- npm 或 pnpm 包管理器

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务器

```bash
npm run dev
```

访问地址：http://localhost:5173

#### 3. 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

#### 4. 预览生产版本

```bash
npm run preview
```

## Docker 一键启动

### 前置要求

- Docker
- Docker Compose

### 启动步骤

#### 1. 构建并启动容器

```bash
docker compose up --build
```

如需后台运行：

```bash
docker compose up --build -d
```

访问地址：http://localhost:5173

#### 2. 停止服务

```bash
docker compose down
```

## 游戏机制

### 纸伞操作

- **开伞**：速度减慢，但受雨水损伤较少，可以滑翔
- **收伞**：速度加快，但受雨水损伤较多

### 障碍物

- **屋檐**：可以避雨，减少伞面损耗
- **水坑**：踩中可短暂加速
- **风口**：会推动角色并损伤伞面
- **石桥**：碰撞会损伤伞面并造成击退
- **灯笼摊**：开伞经过会碰撞，收伞可安全通过
- **码头台阶**：碰撞会损伤伞面
- **狭窄巷道**：开伞经过会碰撞，收伞可安全通过

### 收集品

- **油纸**：恢复 15 点伞面耐久
- **蜡块**：恢复 25 点伞面耐久，并提供 5 秒防水保护
- **金币**：获得 200 分

### 天气系统

- **小雨**：伞面损耗较慢
- **中雨**：正常损耗速度
- **大雨**：伞面损耗较快
- **暴雨**：伞面损耗很快

## 注意事项

- 游戏使用三跑道横向移动设计
- 障碍物出现前会有红色预告提示
- 难度会随时间逐渐增加
- 修补伞面有 8 秒冷却时间
